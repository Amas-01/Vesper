# Task 1.13: Stream Detail Page

Detailed view for individual payment stream with comprehensive information.

## Stream Detail Sections

### Header Section
- Stream ID display
- Status badge (active, completed, cancelled)
- Quick action buttons

### Parties Section
- Sender address with explorer link
- Recipient address with explorer link
- Start and end block numbers
- Created date and time

### Financials Section
- Total deposit amount
- Amount streamed so far
- Amount withdrawn  
- Amount claimable
- Remaining deposit
- Streaming rate per block
